# 🔧 Projet 1 : TrouveTonArtisan
## 📖 Description

Application permettant de rechercher des artisans selon différents critères :
localisation, spécialité, avis clients.

## 🎯 Objectifs
- Proposer une interface intuitive
- Travailler la conception UX/UI
- Réaliser un prototype interactif

## ⚙️ Fonctionnalités
- Recherche et filtrage d’artisans
- Consultation de fiches détaillées
- Formulaire de contact
- Navigation dynamique

## 💡 Points clés
- Conception sur Figma (wireframes + maquettes)
- Développement en Angular
- Responsive design (mobile, tablette, desktop)
- Optimisation SEO et accessibilité

# 📚 Projet 2 : Knowledge
## 📖 Description

Plateforme e-learning permettant de :
- Consulter des cursus et leçons
- Suivre sa progression
- Gérer les contenus (admin)

## 🎯 Objectifs
- Créer une application web complète
- Mettre en place une architecture back-end solide
- Gérer la sécurité et les données utilisateurs

## ⚙️ Fonctionnalités
- Authentification utilisateur
- Gestion des rôles (admin / user)
- CRUD (cours, thèmes, leçons)
- Suivi de progression

## 🔐 Sécurité & RGPD
- Mots de passe hachés (bcrypt / argon2)
- Protection CSRF / XSS
- Gestion des accès
- Minimisation des données
